# SecondServe
SecondServe — Cloud-Native Food Surplus Recovery Platform
A full-stack cloud application built to reduce hawker food waste in Singapore by enabling vendors to list unsold surplus inventory at discounted prices in real time. Built on Microsoft Azure with a FastAPI backend, React PWA frontend, and Azure SQL Database.
Features an AI-powered natural language listing tool (GPT-4o-mini via GitHub Models) that parses Singlish descriptions into structured product data, an automated SGT-aligned expiry engine that clears listings at peak post-meal windows (2:30 PM / 8:30 PM), a Pulse Discount simulation engine, location-based nearby deals discovery, and a customer cart system.
Aligned with the Singapore Green Plan 2030 and Resource Sustainability Act (RSA) compliance requirements.
Stack: Python · FastAPI · React · Azure App Service · Azure SQL · Azure DevOps · CI/CD · GPT-4o-mini · PWA
